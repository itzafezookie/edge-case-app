# EdgeCase: A Big Cube Reference PWA

EdgeCase is a modern web application designed for speedcubers and puzzle enthusiasts who are learning to solve big cubes (4x4 and larger). It serves as a quick and interactive reference for essential algorithms, featuring a clean, mobile-first design that can be installed on any device as a Progressive Web App (PWA) for offline access.

## Live Application

You can access the live application here: **[https://itzafezookie.github.io/edge-case-app/](https://itzafezookie.github.io/edge-case-app/)**

## Features

*   **Algorithm Libraries:** Full libraries for all necessary OLL, PLL, and Parity algorithms, complete with dynamic SVG visualizations that adjust to your selected cube size.
*   **Interactive Commutator Generator:** A unique tool that allows you to select stickers on a cube face to instantly generate the correct commutator algorithm, including complex slice moves.
*   **Dynamic Cube Size:** All visualizations and algorithms dynamically adjust for cube sizes from 4x4 up to 15x15.
*   **Responsive & Mobile-First:** Designed to work beautifully on any screen size, from a desktop monitor to your phone.
*   **PWA Ready:** Installable on your home screen for a native-app feel and complete offline functionality.

## Local Development

To run this project on your local machine:

1.  Clone the repository.
2.  Install the necessary dependencies:
    ```bash
    npm install
    ```
3.  Start the local development server:
    ```bash
    npm run dev
    ```
